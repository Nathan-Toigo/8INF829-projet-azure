# Description du projet

Ce dépôt regroupe les expérimentations réalisées dans le cadre du devoir #1 portant sur les systèmes de **Retrieval-Augmented Generation (RAG)**.

L’objectif principal du projet est d’explorer différentes approches de RAG afin d’évaluer :
- la qualité des réponses générées ;
- la fidélité des informations retournées ;
- l’impact des stratégies de récupération de contexte ;
- les limites et avantages des différentes approches expérimentées.

Chaque membre de l’équipe a réalisé sa propre expérimentation dans un sous-dossier distinct du dépôt.
